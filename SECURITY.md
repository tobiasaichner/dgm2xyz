# Security Policy

`dgm2xyz` processes CAD files supplied by users. Treat drawing files as untrusted input.

## Reporting a Vulnerability

Please report security issues privately to the repository owner instead of opening a public issue.

Include:

- A short description of the issue.
- Steps to reproduce.
- A minimal sample file if it can be shared safely.
- Expected and actual behavior.

## Scope

Relevant security issues include crashes, memory safety problems, unsafe file writes, path traversal, and parser behavior triggered by malformed DXF or future DWG files.
